# Functional Requirements

## Objetivo

Este documento descreve os requisitos funcionais da plataforma PetCare.

Os requisitos funcionais representam os comportamentos esperados do sistema e definem os serviços que deverão ser disponibilizados aos usuários.

Cada requisito possui um identificador único para facilitar rastreabilidade durante análise, desenvolvimento e testes.

---

# Gestão de Usuários

### FR-001 - Cadastro de Usuário

O sistema deve permitir que um usuário realize seu cadastro informando os dados obrigatórios.

---

### FR-002 - Autenticação

O sistema deve permitir que usuários autenticados acessem a plataforma utilizando suas credenciais.

---

### FR-003 - Recuperação de Senha

O sistema deve permitir que o usuário solicite a redefinição de sua senha.

---

### FR-004 - Atualização de Perfil

O sistema deve permitir que o usuário altere suas informações cadastrais.

---

# Gestão de Pets

### FR-005 - Cadastro de Pet

O sistema deve permitir que um tutor cadastre um novo pet.

---

### FR-006 - Atualização de Pet

O sistema deve permitir a edição das informações de um pet cadastrado.

---

### FR-007 - Exclusão de Pet

O sistema deve permitir a remoção de um pet previamente cadastrado.

---

### FR-008 - Consulta de Pets

O sistema deve permitir ao tutor visualizar todos os pets associados à sua conta.

---

### FR-009 - Visualização de Perfil do Pet

O sistema deve permitir visualizar todas as informações registradas sobre um pet.

---

# Carteira de Vacinação

### FR-010 - Registro de Vacina

O sistema deve permitir registrar uma vacina aplicada ao pet.

---

### FR-011 - Atualização de Registro de Vacinação

O sistema deve permitir editar um registro de vacinação.

---

### FR-012 - Exclusão de Registro de Vacinação

O sistema deve permitir remover um registro de vacinação.

---

### FR-013 - Histórico de Vacinação

O sistema deve apresentar o histórico completo de vacinação do pet.

---

# Consultas Veterinárias

### FR-014 - Registro de Consulta

O sistema deve permitir registrar consultas veterinárias.

---

### FR-015 - Atualização de Consulta

O sistema deve permitir editar informações de uma consulta.

---

### FR-016 - Exclusão de Consulta

O sistema deve permitir remover consultas cadastradas.

---

### FR-017 - Histórico de Consultas

O sistema deve apresentar o histórico de consultas veterinárias.

---

# Medicamentos

### FR-018 - Cadastro de Medicamento

O sistema deve permitir registrar medicamentos utilizados pelo pet.

---

### FR-019 - Atualização de Medicamento

O sistema deve permitir editar informações de medicamentos cadastrados.

---

### FR-020 - Exclusão de Medicamento

O sistema deve permitir remover medicamentos cadastrados.

---

### FR-021 - Histórico de Medicamentos

O sistema deve permitir consultar o histórico de medicamentos.

---

# Alergias

### FR-022 - Cadastro de Alergia

O sistema deve permitir registrar alergias do pet.

---

### FR-023 - Atualização de Alergia

O sistema deve permitir editar informações de alergias.

---

### FR-024 - Exclusão de Alergia

O sistema deve permitir remover alergias cadastradas.

---

### FR-025 - Consulta de Alergias

O sistema deve permitir visualizar as alergias registradas para um pet.

---

# Agenda

### FR-026 - Cadastro de Evento

O sistema deve permitir cadastrar eventos relacionados aos cuidados do pet.

---

### FR-027 - Atualização de Evento

O sistema deve permitir alterar eventos cadastrados.

---

### FR-028 - Exclusão de Evento

O sistema deve permitir excluir eventos cadastrados.

---

### FR-029 - Visualização da Agenda

O sistema deve apresentar os eventos futuros relacionados aos pets do tutor.

---

### FR-030 - Lembretes

O sistema deve permitir visualizar lembretes associados aos eventos cadastrados.

---

# Histórico

### FR-031 - Histórico Geral

O sistema deve apresentar uma visão consolidada do histórico de saúde do pet.

---

# Pesquisa

### FR-032 - Pesquisa de Pets

O sistema deve permitir localizar pets cadastrados pelo tutor.

---

### FR-033 - Pesquisa de Eventos

O sistema deve permitir localizar eventos cadastrados.

---

# Segurança

### FR-034 - Controle de Acesso

O sistema deve garantir que um usuário visualize apenas as informações associadas à sua conta.

---

### FR-035 - Encerramento de Sessão

O sistema deve permitir que o usuário encerre sua sessão autenticada.
