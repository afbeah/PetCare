# Wireframes

## Objetivo

Este documento descreve os wireframes da plataforma PetCare.

Seu objetivo é definir a estrutura de cada tela antes da criação da interface visual de alta fidelidade, garantindo consistência entre requisitos, navegação e componentes.

Os wireframes representam a organização da informação e da interação, sem considerar aspectos visuais finais como cores, tipografia ou identidade da marca.

- Desktop
- Tablet
- Mobile

---

# Critérios de Aceitação

Um wireframe será considerado concluído quando:

- representar claramente a hierarquia das informações;
- utilizar os componentes definidos em Components.md;
- respeitar os layouts definidos em Layouts.md;
- atender aos User Flows documentados;
- seguir o Design System da PetCare.
